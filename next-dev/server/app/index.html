Value Investing Criteria

P/E Ratio

Measures stock price relative to earnings. Lower values indicate potentially undervalued stocks.

P/B Ratio

Compares market value to book value. Values below 1 may indicate undervaluation.

Debt/Equity

Measures financial leverage. Lower ratios indicate less financial risk.

ROE

Return on Equity measures profitability efficiency. Higher values indicate better management performance.
